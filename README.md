# Notes app

Aplicativo de notas em VueJS, conhecimentos sobre o framework adquiridos:

  - [x] Diretivas
  - [x] Renderização condicional
  - [x] Renderização de listas
  - [x] v-bind e v-on

Para rodar o aplicativo clone esse repositório e execute os comandos:

```shell
npm i
npm run dev
```
